# Resources Folder

Place your logo file here:

- **File name**: `logo.jpg`
- **Location**: `public/resources/logo.jpg`
- **Usage**: The logo will be accessible at `/resources/logo.jpg` in your app

## Supported formats:
- `.jpg` / `.jpeg`
- `.png`
- `.svg`
- `.webp`

Make sure to name your file exactly `logo.jpg` (or update the path in `constants/index.js` if you use a different name).
